# Дизайн-план сайта психолога Дианы Шутовой

## Визуальный стиль
- **Тип**: Личный сайт психолога
- **Стиль**: Минималистичный, профессиональный, теплый и доверительный
- **Настроение**: Спокойствие, безопасность, компетентность

## Цветовая палитра
- **Основной цвет**: #2C3E50 (темно-синий, для заголовков и акцентов)
- **Вторичный цвет**: #7F8C8D (серо-голубой, для дополнительного текста)
- **Фон**: #FDFEFE (светлый, почти белый, для основного фона)
- **Акцент**: #E8B4B8 (нежный розовый, для кнопок и выделений)

## Типографика
- **Заголовки**: Playfair Display (элегантный serif)
- **Основной текст**: Inter (современный sans-serif)
- **Размер заголовков**: 2.5rem (40px)
- **Размер текста**: 1rem (16px)

## Визуальные эффекты
- Плавная прокрутка (scroll reveal)
- Легкие параллакс эффекты для фонов
- Анимированные карточки услуг
- Интерактивные кнопки с hover эффектами
- Типwriter эффект для приветствия

## Структура сайта
1. **Главная**: Hero секция, краткое представление, призыв к действию
2. **Обо мне**: Детальная информация, образование, подход
3. **Услуги**: Методы работы, стоимость, формат
4. **Контакты**: Способы связи, социальные сети

## Изображения
- Портрет психолога (профессиональное фото)
- Уютный интерьер кабинета
- Природа, абстрактные успокаивающие образы
- Иконки для методов терапии
